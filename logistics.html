Задача

Спроектировать дизайн мобильного приложения для логистов: предоставить выездным сотрудникам данные о заказах, оптимизировать маршруты и автоматизировать отчёты.

Исследование

Изучила открытые кейсы для понимания бизнес-процессов — от приёма заказа до финального отчёта. Проанализировала приложения конкурентов: CDEK для логистов и Яндекс.Доставка для курьеров.

Выводы

  • Распределение заказов по оптимальному маршруту
  • Простой интерфейс с минимальным функционалом
  • Чат с техподдержкой
  • Трекинг для логиста

User Flow

Решения

Авторизация

Вход разрешён только зарегистрированным в CRM пользователям. Авторизация доступна для сотрудников, внесённых в систему.

Главный экран

Помогает логисту быстро оценить загрузку на день, отслеживать сроки доставки и оперативно получать уведомления.

Заказы

Детали заказа, накладная и отправка документов. После завершения доставки логист заполняет чек-лист — время приёмки, состояние товара, подпись контрагента.

Маршрут и навигация

Построение маршрута по всем точкам доставки с возможностью оптимизации порядка адресов и быстрым переходом в навигатор.

Техподдержка

Оперативные ответы от команды техподдержки прямо в приложении.

Сканер

Сканирование штрих-кода для быстрого определения заказов и учёта товаров.
